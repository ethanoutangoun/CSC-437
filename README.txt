Cooked is an interactive website that allows users to post recipes on a community forum, generating popularity for great recipes and allowing creative collaboration between chefs.
To view the website, simply visit https://cooked-ayxh7.ondigitalocean.app/app/.

No need for authentication to view already created recipes, but in order to create, view your account, and change settings, you must be logged in. Authentication is done via JWT, and there are no restrictions for password length/complexity.
